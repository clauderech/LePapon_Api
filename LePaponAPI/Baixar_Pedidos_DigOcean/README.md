# Consumindo Lepapon API

Projeto Python inicial para consumir uma API REST.

## Como executar

1. Instale as dependências:
   ```bash
   pip install requests
   ```
2. Execute o exemplo:
   ```bash
   python main.py
   ```

## Estrutura
- `main.py`: Exemplo de requisição GET para uma API pública.
